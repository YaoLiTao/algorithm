## Yami64

> 一个Base64的实现，在最后两位特殊字符与base64有所不同，
> 可以不经过转义用于http协议。
> 采用无填充方式，每3字节源二进制为一单位，内部分割出每6bit为一编码单元
> 8bit x 3 = 24bit = 6bit x4。

```

'K', 'L', 'M', 'N', 'O', 'P', 'Q', 'R', 'S', 'T',
'U', 'V', 'W', 'X', 'Y', 'Z',

]
```
